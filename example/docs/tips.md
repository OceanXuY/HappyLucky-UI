Use `.hl-tips` as a wrapper element, and add your icons (eg. `.hl-icon-dot`) in it.

{% example html %}
<div class="hl-tips">
  <span class="hl-tips-icon hl-icon-dot"></span> 填写银行卡预留的身份信息
</div>
{% endexample %}

{% example html %}
<div class="hl-tips">
  <span class="hl-tips-icon hl-icon-asterisk"></span> 您可以前往机场指定地点领取行程单
</div>
{% endexample %}
